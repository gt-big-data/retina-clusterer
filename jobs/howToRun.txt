NOTICE: In order to work, this needs to be ran from outside the "retina-clusterer" folder
ALSO: this needs Python >= 2.5 (but other parts need  >= 2.7)


Running the clusterArticles (to assign each article to a category)
---------------------------
python -m retina-clusterer.jobs.1-clusterArticles


Running the keywordExtractor
----------------------------
python -m retina-clusterer.jobs.2-extractKeywords
